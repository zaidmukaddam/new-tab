# new-tab

## Features

- Themes (dark/light)
- Greeting (+ change its position)
- Search
- Weather (+ change its position)

## Installation

### Dev

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`

## Production

1. Install dependencies: `npm install`
2. Package the code: `npm run pkg`
   - To test it, first run `npm run dev:pkg`
3. Done!
